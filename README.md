# Unemployment Analysis in India using Python

## Project Overview
This project analyzes unemployment trends in India using Python, Pandas, and Matplotlib.

## Dataset
The dataset contains:
- Region
- Date
- Unemployment Rate (%)
- Estimated Employed
- Labour Participation Rate (%)
- Area (Rural/Urban)

## Technologies Used
- Python
- Pandas
- Matplotlib

## Analysis Performed
1. Data Cleaning
2. Missing Value Handling
3. Unemployment Rate Distribution
4. Unemployment Trend Over Time
5. Covid-19 Impact Analysis
6. Top 10 States with Highest Unemployment
7. Rural vs Urban Unemployment Comparison

## Key Findings
- Average unemployment increased significantly during Covid-19.
- Tripura and Haryana showed the highest unemployment rates.
- Urban unemployment was higher than rural unemployment.
- April and May recorded the highest unemployment levels.
